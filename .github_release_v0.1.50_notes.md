# v0.1.50

## Highlights / 重點

### Root-Fixed Post-Uninstall Brain-Doc Deadlock / 根治卸載後 Brain Doc 死鎖

`gov_uninstall` now strips governance enforcement rules from **all** brain docs (AGENTS.md, SOUL.md, etc.), not just the bootstrap block. Previously, residual governance rules left in brain docs would block normal workspace operation after uninstall.

`gov_uninstall` 現在會從**所有** Brain Doc（AGENTS.md、SOUL.md 等）中清除治理執行規則，而非僅清除 bootstrap 區塊。此前，卸載後殘留的治理規則會阻礙工作區正常運作。

### Lifecycle Prompt Alignment / 生命週期 Prompt 對齊

- Added Mode B Brain Docs read-only checks to `APPLY_UPGRADE_FROM_BOOT.md`
- Added Mode C Brain Docs write routing and Platform control-plane routing
- 6 new lifecycle content alignment guards in release checker

### UX Branding Refresh / UX 品牌更新

- All `/gov_*` outputs now show branded header, emoji status indicators (✅/⚠️/❌), structured bullet lists, and clear next-step guidance
- Command responses are easier to read; status and next action visible at a glance

### Bug Fix: gov_audit False-Failure / 修復: gov_audit 誤報失敗

- `/gov_audit` no longer reports false failures after `/gov_brain_audit APPROVE`

### Regression Expanded / 回歸測試擴展

- 96/96 (up from 54/54): covers all command modes, user scenarios, flow chains, idempotency, edge cases, and brain-doc residue cleanup

## Install / Update 安裝 / 更新

# New install 新安裝
openclaw plugins install @adamchanadam/openclaw-workspace-governance@latest
openclaw gateway restart

# Existing install update 已安裝更新
