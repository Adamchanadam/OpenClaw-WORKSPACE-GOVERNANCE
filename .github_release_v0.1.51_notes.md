# v0.1.51

## Hotfix / 熱修復

**PLUGIN_VERSION alignment / PLUGIN_VERSION 常數對齊**

v0.1.50 shipped with `PLUGIN_VERSION = "0.1.49"` in `index.ts`, causing branded output headers to display stale version. This patch aligns all three version sources (`package.json`, `openclaw.plugin.json`, `index.ts`).

v0.1.50 出版時 `index.ts` 中的 `PLUGIN_VERSION` 仍為 `"0.1.49"`，導致品牌輸出標頭顯示舊版號。此修補對齊三個版本來源。

No functional change. All v0.1.50 features (brain-doc deadlock fix, lifecycle alignment, 96/96 regression) are included.

無功能變更。v0.1.50 所有功能（Brain Doc 死鎖根治、生命週期對齊、96/96 回歸測試）均已包含。

## Install / Update 安裝 / 更新

```bash
openclaw plugins update openclaw-workspace-governance
openclaw gateway restart
```

Then in OpenClaw TUI: `/gov_setup quick`
