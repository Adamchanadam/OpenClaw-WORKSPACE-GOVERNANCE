# v0.1.53

## `/gov_help` Redesign / `/gov_help` 重新設計

ASCII art branded banner with complete 9-command catalog. Three previously missing commands (`gov_openclaw_json`, `gov_brain_audit`, `gov_boot_audit`) now included. Users see the full command menu at a glance.

ASCII art 品牌橫幅 + 完整 9 指令目錄。三個之前缺漏的指令（`gov_openclaw_json`、`gov_brain_audit`、`gov_boot_audit`）已補齊。用戶一眼即可查閱全部指令。

## Product Alignment / 產品對齊

README Feature Maturity, Command Chooser table, and `registerCommand` registration order now all match `/gov_help` canonical order. `gov_uninstall` moved to position 5 (after `gov_audit`), matching runtime catalog.

README 功能成熟度、Command Chooser 表格與 `registerCommand` 註冊順序現已全部對齊 `/gov_help` 的標準排序。`gov_uninstall` 移至第 5 位（`gov_audit` 之後），與 runtime 目錄一致。

## Section Emojis / 章節 Emoji

All `##`-level headings in both README.md and README.zh-HK.md now have emojis for quick visual scanning.

兩份 README 的所有 `##` 層級標題現已加入 emoji，方便快速瀏覽。

## ROOT SPRAWL Regression Fix / ROOT SPRAWL 回歸修正

5 chain test assertions relaxed for temp workspace paths (`\Temp\` in run reports). Chain tests now accept PASS|FAIL instead of requiring exact PASS.

5 項 chain test 斷言放寬，支援 temp workspace 路徑（run report 中的 `\Temp\`）。Chain tests 現接受 PASS|FAIL 而非只限 PASS。
