# v0.1.62 — Runner Defensive Fixes + Task Context Continuity

## What's New

### Phase 1: Task Context Continuity (C169-C178)
- All PASS-status command outputs now include task-return hint ("Continue with your current task")
- Post-update reminder uses task-first framing instead of demanding immediate governance action
- Command descriptions cleaned of internal jargon

### Phase 2: Runner Defensive Fixes (C179-C183)
- **QC #8 uninstall awareness**: audit now treats "backup exists + target absent" as confirmed removal (PASS) for uninstall reports
- **workspace_root mismatch warning**: audit result includes diagnostic when latest write report used a different workspace_root
- **TOCTOU protection**: readFileSync calls wrapped in try-catch across audit and migrate runners (3 call sites + 1 in new code)
- **Customization overwrite warning**: upgrade mode now reports which files had user customizations overwritten
- **Corrupted config detection**: alignAllowlistEntry returns CONFIG_CORRUPTED instead of silently replacing broken JSON
- **Backup failure protection**: upgrade blocked with BACKUP_FAILED if backup copy throws
- **Concurrent race documentation**: known limitation documented in gov_setup
