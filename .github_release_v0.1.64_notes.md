# v0.1.64 — Pre-Publish Machine Guard (README Release Notes)

## What Changed
- `check_release_consistency.mjs` now enforces that both README.md and README.zh-HK.md release notes tables contain the current `package.json` version
- Consistency check FAILs if README release notes are stale — prevents publishing without documentation update
- AGENTS.md §7b updated with mechanism documentation and v0.1.63 lesson

v0.1.63 lesson: README release notes table was not updated before publishing v0.1.62, forcing a patch release because npm does not allow republishing the same version. This machine guard ensures it cannot happen again.
