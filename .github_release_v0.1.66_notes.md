# v0.1.66 — Cron Read/Write Split + Heartbeat Write Governance

## What Changed

**Problem**: `openclaw cron add/update/remove/pause/resume` and `openclaw gateway heartbeat` config changes bypassed ALL governance gates. The agent could modify cron schedules and heartbeat config without reading official documentation first.

**Fix**: Split cron commands into read (bypass OK) and write (require Mode C governance). Added heartbeat write detection. Official doc URLs are injected as route hints so the agent reads docs before making changes.

### Cron Command Classification

| Command | Classification | Mode C |
|---------|---------------|--------|
| `openclaw cron list/ls/show/status/run/runs` | Read | Bypass (no governance gate) |
| `openclaw cron add/update/remove/delete/pause/resume` | Write | Required — PLAN→READ→CHANGE→QC→PERSIST |
| `openclaw cron` (bare, no subcommand) | Read | Bypass |
| `openclaw gateway heartbeat` config changes | Write | Required |

### Doc URL Route Hints

- Cron writes → https://docs.openclaw.ai/automation/cron-jobs
- Heartbeat writes → https://docs.openclaw.ai/gateway/heartbeat

### Regression

190 → 197/197 (+7 new tests: cron-read-bypass, cron-write-blocked, cron-write-not-bypass, heartbeat-write-modeC, cron-write-doc-hint, heartbeat-write-doc-hint, cron-bare-read)

## Upgrade 升級

```bash
# Existing install update 已安裝更新
openclaw plugins update openclaw-workspace-governance
openclaw gateway restart
```

Then in OpenClaw: `/gov_setup quick`

# v0.1.66 — Cron 讀寫分離 + Heartbeat 寫入治理

## 變更內容

**問題**：`openclaw cron add/update/remove/pause/resume` 及 `openclaw gateway heartbeat` 配置變更繞過所有治理閘門。Agent 可在未閱讀官方文檔的情況下修改排程及心跳配置。

**修復**：將 cron 指令分為唯讀（繞過）和寫入（需 Mode C 治理）。新增 heartbeat 寫入偵測。官方文檔 URL 注入為路由提示，確保 agent 修改前先讀取文檔。

### Cron 指令分類

| 指令 | 分類 | Mode C |
|------|------|--------|
| `openclaw cron list/ls/show/status/run/runs` | 唯讀 | 繞過（無治理閘門） |
| `openclaw cron add/update/remove/delete/pause/resume` | 寫入 | 必須 — PLAN→READ→CHANGE→QC→PERSIST |
| `openclaw cron`（裸指令，無子命令） | 唯讀 | 繞過 |
| `openclaw gateway heartbeat` 配置變更 | 寫入 | 必須 |

### 文檔 URL 路由提示

- Cron 寫入 → https://docs.openclaw.ai/automation/cron-jobs
- Heartbeat 寫入 → https://docs.openclaw.ai/gateway/heartbeat

### 回歸測試

190 → 197/197（+7 新測試：cron-read-bypass、cron-write-blocked、cron-write-not-bypass、heartbeat-write-modeC、cron-write-doc-hint、heartbeat-write-doc-hint、cron-bare-read）

## 升級

```bash
# 已安裝更新
openclaw plugins update openclaw-workspace-governance
openclaw gateway restart
```
